# WholeSalePosWithJPA

### Installation

1. Initialize git repository
```sh
git init
```
2. Clone the repo
```sh
git clone https://github.com/AnujaKoralage/WholeSalePosWithJPA.git
```
3. Import project via Intellij Idea

## Prerequisites

+ Intellj idea
+ Jasper Studio
+ JavaSE(JavaFX)
+ SceneBulider

## Built With

+ JavaFX
+ Java
+ Java Persistence API

## Technological Stack

- JPA
- JavaFX
- JDBC
- Jasper
- MySql
